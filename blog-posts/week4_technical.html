Going for the Cycle

For today's blog I decided to look at the Enumberable 'cycle' if for no other reason than the fact that I just had never heard of it. I'm glad I chose it because it's a really interesting method and has the potential to be used in a variety of problems I might face when coding in Ruby.

Basically cycle can be used on an array and can pass an optional block of code that does something to each element passed to the block. When the last item has been passed to that block, cycle starts over at the beginning of the array and goes through again. This is basically an infinite loop method unless you include a break with some type of condition built into the block.

When researching this subject I came across another blog post by Joey deVilla on his tech blog Global Nerdy which explained very well how this method works. In the blog he uses the example of using cycle on days of the week, and that using Enumerable's "next" method can give us the next day of the week.

The best example to show how cycle works is when you're given two lists that you want to match, but the problem is that the lists do not have an equal number of elements. Lets say we have 3 security guards and we want them to work 7 days a week in a cycle. The cycle method helps us with that.

guards = ["Curly", "Larry", "Moe"]
days_of_the_week = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"]

security = guards.cycle
days = days_of_the_week.cycle

Once we've created our variables that cycle through our guard names and the days of the week we can use a times method to assign the guards to the days of the week in the first week. First of all we will need to call the method 'rewind' to make sure we are starting with the first guard

security.rewind

7.times do   guard = security.next
  day = days.next
  puts "On #{day}, Guard #{guard} will be on duty"
end

=> On Monday, Guard Curly will be on duty
On Tuesday, Guard Larry will be on duty
On Wednesday, Guard Moe will be on duty
On Thursday, Guard Curly will be on duty
On Friday, Guard Larry will be on duty
On Saturday, Guard Moe will be on duty
On Sunday, Guard Curly will be on duty

As you can see, each guard has now been assigned to a day of the week and it can be done continuously. Remember it is very important that you either include the cycle method as part of a loop with a built in break such as the 'times' method or to write in a break that occurs when a certain condition is fulfilled.

I'd love to think of new and interesting ways to implement the cycle method, but for now it's already a very interesting method to use in your box of Ruby tools. Enjoy!
