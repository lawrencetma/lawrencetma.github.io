How Does the Internet Work?

I type in www.google.com and the most popular search engine pops onto my screen. But how did this magic occour?

The first thing to remember is that all computers similar to a house, has an address. In the computer world, that address is called an IP Address. Ok you probably already knew that, but did you know that the website you are visiting also has an IP address?

For an experiment conduct a reverse IP Domain Check here

Once you get that IP address, copy it and paste it into your navigation bar and see the same page pop up. Amazing!

But we don't type in IP addresses in order to get to webpages, how does entering in a web address translate to an IP address?

First, your modem is what connects your computer to the internet. A modem is hooked up between your computer and the internet through cables. The modem is there to translate the signal coming in from those cables so your computer can put them up on your screen.

Your computer through your modem will then send a message to a DNS server or Domain Name Server.

Domnain Name Servers can be thought of as a re-routing service. The DNS servers have lists and lists of different websites and what they have is the domain name of the website such as www.google.com and the IP address of that website. Therefore when you type in www.google.com, your computer sends a packet of information to the DNS server in your area and asks it to forward to the correct IP address of that domain name.

That packet of information contains your own IP address so that the web server knows where to send the information back.The DNS server looks up www.google.com, sees the corresponding IP address and sends your request to google's servers.

Once your request is sent to the server, that server then responds with the corresponding web page which is then sent directly back to you as a packet of information that will allow your computer to display the entire web page.

And viola! We have internet! This entire operation probably took you a matter of seconds if not less, but it's fascinating how it occurs. There are some other complexities but for the most part, this is how you are connected to the internet. Now go test it out!
